# crypti-cli

A command line interface for bootstrapping and managing Crypti DApps.

Requires:

* Crypti 0.5.0

## Installation:

To install run:

```sh
npm install -g crypti-cli
```

Or if you are not root:

```sh
sudo npm install -g crypti-cli
```

## Help

Run to get help:

```sh
crypti-cli -h
```

## DApps

### Add DApp

```sh
crypti-cli dapps -a
```

### Deposit

```sh
crypti-cli dapps -d
```

### Withdraw

```sh
crypti-cli dapps -w
```

## Contracts

### Add contract

```sh
crypti-cli contract -a
```

### Delete contract

```sh
crypti-cli contract -d
```

## Crypto

### Generate public key

```sh
crypti-cli crypto -p
```

### Generate accounts

```sh
crypti-cli crypto -g
```
